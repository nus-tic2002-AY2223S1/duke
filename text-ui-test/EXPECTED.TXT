| S/N | Command                                         | Description                           | Usage Example                                |

| 1   | deadline <task> /by <date & time> /<periodical> | Add deadline with date and periodical | deadline taskABC /by 2022-12-31 10:30 /daily |
| 2   | event <task> /at <date & time> /<periodical>    | Add event with date and periodical    | event taskABC /at 2022-12-31 10:30 /monthly  |
| 3   | todo <task> /<periodical>                       | Add todo task and periodical          | todo taskABC                                 |
| 4   | list                                            | List existing tasks                   | list                                         |
| 5   | mark <task_no>                                  | Mark specified task                   | mark 1                                       |
| 6   | unmark <task_no>                                | Unmark specified task                 | unmark 1                                     |
| 7   | delete <task_no>                                | Delete specified task                 | delete 1                                     |
| 8   | find <keyword>                                  | Find keyword in task list             | find meeting                                 |
| 9   | update <task_no> desc/date <task_details>       | Update task by desc/date              | update 1 date 2022-12-31 10:30               |
| 10  | bye                                             | Exit duke bot                         | bye                                          |
-----------------------------------------------------------------------------------------------------------------------------------------------
Notes:
1. <date & time> should be in the format of yyyy-mm-dd HH:mm.
2. update command allows the update of both description and datetime by stating `desc` and `date` respectively as part of the input.
3. /<periodical> is optional for applicable commands.
4. <periodical> consists of daily, weekly, monthly, quarterly, biannually, annually
